Getting to know Github

How to push your local repo to Github and then pull it to your remote repo? >

Adding empty project to GitHub

1) Get inside where your project files will be kept

2) Check if git is already initialized: git status

  • If it is not being tracked by git, you must initialize it: git init

    • 3) Create new repository on your Github account.

    4) Connect your local project folder to your empty folder/repository on Github.

  • Copy the link of your repo on GitHub
  • Go back to your project in the terminal
  • Type: git remote add origin [link of your repo]
  • Push your branch to Github: git push origin master

    • Voila! You should see your project files in Github repository

    Adding an exisiting project to GitHub

    1) Get inside where your project files are kept

    2) Check if git is already initialized: git status

  • If it is not being tracked by git, you must initialize it: git init

    • 3) Add the files in your new local repository. This stages them for the first commit: git add OR git add . (to add all files)

    4) Commit the files that you've staged in your local repository: git commit -m "comment."

    5) Copy the link of your repo on GitHub

    6) In terminal, add the URL for the remote repository where your local repository will be pushed

  • git remote add origin [link of your repo]
  • git remote -v

    • Push the changes in your local repository to GitHub: git push origin master
